# Local Development

A small Vue 3 + Vite project.

Requirements

- Node.js matching the engines in [package.json](package.json) (recommended: Node 20+)

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
